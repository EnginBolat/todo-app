## Run Locally

Clone the project

```bash
  git clone https://github.com/EnginBolat/todo_app.git
```

Go to the project directory

```bash
  cd todo_app
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
